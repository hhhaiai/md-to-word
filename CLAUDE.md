# 项目状态记录 - CLAUDE.md

## 项目概述

Markdown到Word公文格式转换工具，符合GB/T 9704-2012《党政机关公文格式》国家标准。

## 已完成功能

### ✅ 核心转换功能
- [x] 基本Markdown到Word转换
- [x] 公文格式设置（页边距、字体、字号、行距）
- [x] 标题层级处理
- [x] 页码添加
- [x] 命令行接口

### ✅ 格式处理
- [x] YAML front matter过滤
- [x] 结尾元数据过滤（Date、标签等）
- [x] 加粗标记去除（**文字**、__文字__）
- [x] 数学公式处理（LaTeX转纯文本）
- [x] 成文日期汉字转换
- [x] 附件说明格式化

**现象**：Word文档中出现异常的字符间距，文字被不正常分割
**原因**：Markdown源文件中存在不正常的换行导致
**状态**：需要修复
**位置**：
- markdown_parser.py中的_merge_broken_lines()方法
- word_generator.py中的段落处理逻辑

**示例问题**：
```
载流子浓度达1.78× 10^18
     cm⁻³
```
应该连续显示，但在Word中被异常分割。

## 技术栈

- **Python 3**: 主要编程语言
- **python-docx**: Word文档操作库
- **markdown**: Markdown解析库
- **re**: 正则表达式处理

## 使用流程

1. **输入**：Markdown文件（支持任意路径）
2. **解析**：
   - 过滤YAML front matter
   - 去除加粗标记
   - 处理数学公式
   - 合并分割行（有问题）
   - 过滤结尾元数据
3. **转换**：
   - 设置页面格式
   - 添加文档标题（使用文件名）
   - 处理各级标题
   - 格式化正文段落
   - 添加附件说明
   - 添加成文日期
   - 设置页码
4. **输出**：符合公文格式的Word文档
